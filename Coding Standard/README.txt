To import coding standard:
1) Run Eclipse:
1a) select from menu:  Window => Preferences
1b) select from tree:  Java => Code Style => Formater
1c) Press Import button
1d) Select eclipseFormater.xml from Coding Standard
1e) Press Apply button

Set cleanUp. That format your code when you press Ctrl+S.
2a) select from menu:  Window => Preferences
2b) select from tree:  Java => Code Style => Clean UP
2c) Press Import button
2d) Select cleanUp.xml from Coding Standard
2e) Press Apply button
2f) select from tree:  Java => Editor => Save Actions
2g) set Perform the selected actions on save
2h) set Format source code, select Format all lines
2i) set Organize imports

The end.
